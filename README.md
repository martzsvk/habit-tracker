# CLI Habit Tracker

Simple command-line program built in Python. The application allows users to add habits and store them in a JSON file
so the data persists between runs.

## Features
- Add new habits
- View saved habits
- Persistent storage using JSON
- Simple CLI menu

## How to run
1. Clone the repository
2. Navigate into the project folder
3. Run:

```bash
python main.py
```
