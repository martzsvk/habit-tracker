# CLI Habit Tracker

A simple command-line Habit Tracker built in Python. The application allows users to add habits and stores them in a JSON file
so the data persists between runs.

## Features
- Add new habits
- View saved habits
- Persistent storage using JSON
- Marking habits as completed
- Simple CLI menu

## How to run
1. Clone the repository
2. Navigate into the project folder
3. Run:

```bash
python main.py
```
